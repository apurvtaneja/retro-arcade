# Retro Arcade Website

A retro 80s arcade-style interactive website featuring multiple classic games with neon aesthetics, pixel art styling, and CRT scanline effects.

## Features

- **4 Classic Arcade Games:**
  - Snake - Classic snake game with food collection
  - Pong - Single-player vs AI paddle game
  - Tetris - Falling blocks puzzle game
  - Space Invaders - Side-scrolling shooter

- **Retro 80s Aesthetic:**
  - Neon cyan, magenta, and yellow color scheme
  - Pixel font (Press Start 2P)
  - Animated CRT scanlines
  - Glowing neon text effects
  - Arcade cabinet design

## Controls

### Snake
- Arrow Keys: Move
- ESC: Return to menu

### Pong
- W/S: Move paddle up/down
- ESC: Return to menu

### Tetris
- Arrow Left/Right: Move piece
- Arrow Down: Move piece down faster
- A/D: Rotate piece
- ESC: Return to menu

### Space Invaders
- Arrow Left/Right: Move player
- Space: Shoot
- ESC: Return to menu

## How to Play

1. Open `index.html` in a web browser
2. Select a game from the menu
3. Play the game
4. When the game ends (win or lose), the end screen will appear
5. Click "RETURN TO MENU" to play another game

## Technical Details

- Pure vanilla JavaScript (no frameworks)
- HTML5 Canvas for game rendering
- CSS animations for visual effects
- Responsive design for different screen sizes
